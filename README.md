# FORTALHUB

Aplicativo mobile desenvolvido para ajudar cidadãos a localizar equipamentos públicos próximos na cidade.

## 📱 Objetivo

O objetivo do aplicativo é facilitar o acesso da população a serviços públicos, permitindo identificar rapidamente o equipamento mais próximo da sua localização.

## 🏥 Equipamentos disponíveis

O aplicativo permite localizar:

- Hospitais
- Equipamentos esportivos
- Ecopontos

## 📍 Funcionalidades

- Identificação do equipamento público mais próximo
- Listagem de todos os equipamentos disponíveis
- Uso do GPS do dispositivo
- Interface simples e intuitiva

## 🛠 Tecnologias utilizadas

- React Native
- Expo
- JavaScript
- Android

## 📊 Impacto Social

O aplicativo contribui para melhorar o acesso da população aos serviços públicos da cidade, facilitando a localização de equipamentos importantes.
